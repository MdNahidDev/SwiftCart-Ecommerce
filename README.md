1) What is the difference between null and undefined?

undefined =========> ভেরিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু কোনো ভ্যালু দেওয়া হয়নি।
null =========> ইচ্ছাকৃতভাবে “খালি” মান সেট করা হয়েছে।

2) What is the use of the map() function in JavaScript? How is it different from forEach()?

map() =========> প্রতিটি আইটেমে কাজ করে এবং নতুন অ্যারে রিটার্ন করে।
forEach() =========> শুধু লুপ চালায়, কিছু রিটার্ন করে না।

3) What is the difference between == and ===?

==  =========> শুধু ভ্যালু চেক করে
===  =========> ভ্যালু , টাইপ দুটোই চেক করে।

4) What is the significance of async/await in fetching API data?

API থেকে ডেটা আনতে সময় লাগে।
async/await কোডকে সহজ ও পরিষ্কার করে।
await ডেটা না আসা পর্যন্ত অপেক্ষা করে।

5) Explain the concept of Scope in JavaScript (Global, Function, Block).

Scope মানে ভেরিয়েবল কোথায় ব্যবহার করা যাবে।
Global Scope =========> সব জায়গা থেকে ব্যবহার করা যায়।
Function Scope =========> শুধু ফাংশনের ভিতরে।
